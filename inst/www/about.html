About

This is a prototype website showcasing an original Geographic information analytics (GiX)
app, based on: OpenCPU (R, PostgreSQL/PostGIS, D3 and Leaflet). Basemap by Mapbox.
GeoTuple provides basic exploratory data analysis tools that let's you view patterns
and discover relationships in geospatial data.

Visit geotuple.org for more info.
